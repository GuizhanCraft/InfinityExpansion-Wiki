# 分解机

分解机是可以把部分方块还原成基础材料的机器。

## 等级

目前分解机有1个等级。

**电力消耗**: 120 J/s

## 配方

下表列出了分解机可用的配方。

| 输入 | 输出 |
| ---- | --- |
| 绿宝石块 | 绿宝石 * 9 |
| 金块 | 金锭 * 9 |
| 铁块 | 铁锭 * 9 |
| 下界合金块 | 下界合金锭 * 9 |
| 红石块 | 红石粉 * 9 |
| 石英块 | 下界石英 * 9 |
| 青金石块 | 青金石 * 9 |
| 煤块 | 煤 * 9 |
| 五重压缩的石头 | 四重压缩的石头 * 9 |
| 四重压缩的石头 | 三重压缩的石头 * 9 |
| 三重压缩的石头 | 双重压缩的石头 * 9 |
| 双重压缩的石头 | 简易压缩的石头 * 9 |
| 简易压缩的石头 | 圆石 * 9 |
